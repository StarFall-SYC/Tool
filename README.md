别担心！这个工具合集就是为解决这些烦恼而生的！

## 🎯 现有工具

### 📝 文本缩进工具 (text_indent.py)
- 一键搞定文本缩进
- 支持批量处理文件夹
- 可自定义缩进空格数
- 贴心的预览功能

### 📂 文件重命名工具 (Rename_files.py)
- 批量重命名，效率upup！
- 支持多种命名规则
- 可以添加前缀、后缀
- 贴心的预览和撤销功能

### 🔑 密钥生成器 (key_generator.py)
- 生成安全的随机密钥
- 多种格式可选

### 💬 微信轰炸器 (WeChatBoom.py)
- 自动发送消息
- 请谨慎使用，后果自负 😏
